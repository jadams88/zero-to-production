Zero to Production

Fullstack Monorepo Starter

About

A full stack Monorepo starter to use for the foundation for your next project.

Guides

Follow the guides to step through deploying your own UI with Firebase and your API either serverless with AWS Lambda or in a Kubernetes Cluster on Google Cloud.

Whats in the box

  • Two Angular apps. This "demo" site and a Todo App with authentication using NgRx & GraphQL.
  • A Node.js GraphQL API with Koa.js , GraphQL & MongoDB configured to deploy serverless or with Docker & Kubernetes.
  • Guides to get you up and running with a deployed Angular application and API server.

Angular Demo

Demo of some less more advanced Angular techniques.
